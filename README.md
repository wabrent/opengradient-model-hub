# OpenGradient Model Hub

Interactive AI model explorer for OpenGradient Network.

## Features

- Search models by name and type
- zkML and TEE verification badges
- Clean, responsive design

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- TypeScript

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically
