# OpenGradient Model Hub

Interactive AI model explorer for OpenGradient Network with verifiable computing.

## Features

- 🔍 Search models by name, type, or category
- 🎯 Filter by model type (LLM, NLP, Computer Vision, etc.)
- 🔐 Filter by verification (zkML, TEE)
- 📊 Network statistics dashboard
- 📱 Fully responsive design
- 🖱️ Click on any model to see detailed information

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- TypeScript

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel dashboard
3. Deploy automatically

## Models Included

12 AI models across different categories:
- LLM: LLaMA-7B, Mistral-7B, CodeLlama-13B, Falcon-40B
- NLP: GPT-2 Sentiment, BERT Base
- Computer Vision: ResNet-50, YOLOv8, ViT Base
- Audio: Whisper Small
- Generative: Stable Diffusion XL
- Multimodal: CLIP
